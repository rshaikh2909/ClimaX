# 🌤 ClimaX - Weather Dashboard

A modern, responsive weather dashboard built using HTML, CSS, and JavaScript. ClimaX provides real-time weather information, hourly and 7-day forecasts, weather animations, and a clean glassmorphism user interface.

---

## 🚀 Live Demo

🔗 (https://clima-x-gamma.vercel.app/)

---

## ✨ Features

- 🌍 Search weather by city
- 📍 Current location weather
- 🌤 Real-time weather updates
- 📅 24-hour forecast
- 📆 7-day forecast
- 🌧 Dynamic weather animations
- 🌙 Dark / Light mode
- 📱 Fully responsive design
- 🎨 Modern Glassmorphism UI
- ⚡ Fast loading experience

---

## 🛠 Technologies Used

## 🔧 Installation

1. Clone the repository

```
git clone https://github.com/yourusername/climax-weather.git
```

2. Open the project folder.

3. Add your WeatherAPI key in:

```
js/api.js
```

4. Open `index.html` in your browser.
