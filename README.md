# 🌤️ ClimaX - Modern Weather Dashboard

A modern, responsive, and interactive **Weather Dashboard** built using **HTML, CSS, and JavaScript**. ClimaX provides real-time weather information, forecasts, location-based weather, and beautiful weather animations using the **WeatherAPI**.

## ✨ Features

* 🌍 Search weather by city
* 📍 Get weather using current location (Geolocation API)
* 🌡️ Real-time temperature
* 🌤️ Weather description
* 💧 Humidity
* 🌬️ Wind Speed
* 🌫️ Visibility
* 📊 Air Quality Index (AQI)
* ☀️ UV Index
* 🌅 Sunrise & Sunset
* 🕒 Last Updated Time
* ⏰ 24-Hour Weather Forecast
* 📅 7-Day Weather Forecast
* 🌙 Dark Mode
* 🎨 Glassmorphism User Interface
* ☀️ Animated Sun
* ☁️ Moving Clouds
* 🌧️ Rain Animation
* ❄️ Snow Animation
* ⚡ Lightning Animation
* 🌌 Dynamic Weather Backgrounds
* 📱 Responsive Design

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/ClimaX.git
```

2. Open the project folder.

3. Add your WeatherAPI key inside `api.js`.

```javascript
const API_KEY = "YOUR_API_KEY";
```

4. Open `index.html` using **Live Server**.
